# Student Grade Management System

## Overview
A Java console-based application that manages student records and calculates GPA using object-oriented programming principles.

## Features
- Add courses with grades and credit hours
- Calculate weighted GPA
- Display full academic record
- Modular class-based structure

## Technologies Used
- Java
- OOP (Encapsulation, Classes, Objects)
- ArrayList
- Eclipse IDE

## How to Run
1. Compile all Java files
2. Run Main.java
3. Follow console prompts
